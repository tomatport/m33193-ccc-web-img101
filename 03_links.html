Making Links

To turn any image into a link, wrap it in an anchor element (i.e. <a href=""><img src="" alt=""></a>)

Below, each teapot is wrapped in a link to a fragment URL. Each fragment URL addresses an element within this document.

Notice that:

  • The fragment appears at the end of the URL in the address bar and changes with each click.
  • In the css a :target pseudo selector is used to color the background of the element with the same ID as the fragment.
  • If necessary, the document is scrolled until the target is visible.
  • The anchor that wraps the university logo has a longer URL that contains the full http://port.ac.uk address. Anchors can point to parts of the current document, or link to any other document on the web.

Look at each of the CSS rules. Try to deduce:

  • The purpose of each selector.
  • What each property affects - if you're not sure, try commenting out each property to see what changes.
