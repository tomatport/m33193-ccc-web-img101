  • Observe the figure element which may contain an image and a figcaption element.
  • Figure can contain other things, for example, a table of data in an academic paper usually has a caption.
  • figure elements may be styled with border, margins, colours, floats, etc., like any other elements.
  • The flex display property allows us to change the orientation of the image and text with the flex-direction property.
  • The use of two @media queries provides for different layouts when the page is wide (landscape orientation) or narrow (portrait orientation).
