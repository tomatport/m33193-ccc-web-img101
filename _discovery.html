Answers to Questions

01_img.html

What happens if newlines are added/removed within the element?

They are ignored.

What happens when the src url is changed?
  • Try src="800x200", src="800x800", or something unexpected like src="eSp1nQ".

The sources that exist are displayed, but when the source does not exist, the alt text is displayed instead.

What happens if quotation marks are removed from around the src attribute?

As long as the source contains no spaces, it will still work. Otherwise, the browser will not know where the source ends and the next attribute begins.

What happens if quotation marks are removed from around the alt attribute?

As with the src attribute, the browser will not know where the attribute ends and the next attribute begins if there are spaces present in the alt text.

How might the title attribute help us?

Browsers will display the title text when the mouse is hovered over the image, and screen readers will read the title text aloud.

02_size_css.html

What happens when both width and height CSS properties are used simultaneously? Does this image look 'right'?

The aspect ratio of the image is not maintained, so the image looks stretched.

What happens if a second image is added to the page? Are both images formatted the same?

In this case, since the img selector is used, both images are formatted the same.

What happens if the font-size in the body rule is changed to 2em, 3em or larger?

The image will be larger, since the width is defined in rems, which are relative to the font size.

What alternative sizing units are there?

% is relative to the parent element, em is relative to the font size, and vmin is relative to the viewport size. vmin will choose the smallest of the viewport's height or width, so it is useful for ensuring that an image will fit on the screen.
